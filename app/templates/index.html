Test Design Tools

T-way Covering Array

Generate minimal test suites that cover all t-way combinations of factor levels. Supports pairwise (t=2), three-way (t=3), and beyond.

Orthogonal Array Analyser

Enter an array and verify its OA properties: OA(N,k,s,t) notation, index λ, L notation, and strength checks.

Mixed OA Analyser

Analyse mixed-level (asymmetric) orthogonal arrays. Define column groups sk, set strength t, and verify MOA properties.

Fault Analysis

Determine the interaction strength t of test cases that result in faults. Identify which combinations are unique to failing tests.
